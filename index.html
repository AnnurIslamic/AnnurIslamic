Kebijakan Privasi

Pembaruan Terakhir: 4 Juli 2025

Aplikasi Annur Islamic ("kami" atau "milik kami") berkomitmen untuk melindungi privasi Anda. Kebijakan Privasi ini menjelaskan bagaimana kami mengumpulkan, menggunakan, dan melindungi informasi Anda saat Anda menggunakan aplikasi kami.


Pengumpulan Informasi Lokasi:

Aplikasi ini mengumpulkan data lokasi geografis Anda (latitude dan longitude) hanya untuk tujuan menghitung waktu sholat yang akurat. Data ini tidak pernah disimpan di server kami atau dibagikan dengan pihak ketiga mana pun. Semua perhitungan dilakukan secara lokal di perangkat Anda.


Penyimpanan Lokal:

Kami menggunakan penyimpanan lokal (localStorage) di browser Anda untuk menyimpan preferensi seperti tema (mode malam), bahasa, dan data cache jadwal sholat untuk penggunaan offline. Data ini tetap berada di perangkat Anda dan tidak dikirim ke tempat lain.

‹

Ketentuan Penggunaan

Dengan mengunduh atau menggunakan aplikasi Annur Islamic, Anda secara otomatis menyetujui ketentuan berikut:

  • Anda tidak diizinkan untuk menyalin, atau memodifikasi aplikasi, bagian mana pun dari aplikasi, atau merek dagang kami dengan cara apa pun.
  • Aplikasi ini disediakan "sebagaimana adanya". Meskipun kami berusaha untuk memastikan keakuratan data (seperti waktu sholat dan konten Al-Qur'an), kami tidak memberikan jaminan atas keakuratannya. Pengguna disarankan untuk melakukan verifikasi silang dengan sumber tepercaya.
  • Kami berhak untuk mengubah ketentuan ini kapan saja. Penggunaan aplikasi secara berkelanjutan setelah perubahan akan dianggap sebagai penerimaan Anda terhadap ketentuan baru.
‹

Informasi Tambahan

Sumber Data:

  • Waktu Sholat: Menggunakan API dari Aladhan.com dengan metode kalkulasi ISNA (Islamic Society of North America).
  • Al-Qur'an: Teks, terjemahan, dan data surah diambil dari API publik Quran.gading.dev.
  • Lokasi: Nama kota didapatkan melalui layanan Nominatim (OpenStreetMap).

Aplikasi ini adalah proyek non-komersial dan dibuat untuk tujuan membantu umat Muslim dalam menjalankan ibadah sehari-hari. Aplikasi ini bersifat gratis dan tidak mengandung iklan.
